AutoShop Pro - Phase 1

Included
- Firebase Authentication (email/password)
- Signup + login
- User role stored in Firestore
- Customer CRUD
- GitHub/Vercel-ready structure

Setup
1. Create Firebase project.
2. Enable Authentication > Email/Password.
3. Create Firestore.
4. Copy firebase-config.example.js to firebase-config.js
5. Paste your Firebase config.
6. Deploy on Vercel or open locally.
